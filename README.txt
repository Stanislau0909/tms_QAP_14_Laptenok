Описание команд для установки проекта из git
1)Скачать репозиторий из github с помощью команды git clone <URL>
2)Создать ветку с помощью команды git checkout -b <NAME_BRANCH>
3)Внести изменения в файле или создать новый файл и написать свой код или автотест
4)Что бы проиндексировать все изменения надо использовать команды git add .
5)Сделать коммит    git commit -m "comment"   в комментарии оставить коммент связанный с изменением
6)сделать git push origin QAP_14
